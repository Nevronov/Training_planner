# Планировщик тренировок

Планировщик тренировок — это веб-приложение для планирования спортивных тренировок и отслеживания физического прогресса. Проект представляет собой полноценный дневник, который позволяет пользователям создавать собственные программы тренировок, вести базу упражнений и следить за изменениями своего веса.

Основной функционал

Система аккаунтов: Регистрация и авторизация пользователей, безопасное хранение паролей (хеширование через bcrypt + sha256) и авторизация на базе Cookie-сессий.
Конструктор тренировок: Создание, редактирование и удаление шаблонов тренировок. Возможность задавать индивидуальное количество подходов и рабочий вес для каждого упражнения.
База упражнений: Просмотр встроенных упражнений с разделением по группам мышц, а также возможность добавлять собственные (кастомные) упражнения.
Календарь занятий: Планирование тренировок на определенные дни и удобная система отметок об их выполнении.
Профиль и статистика: Отслеживание истории изменения собственного веса и просмотр общей статистики по выполненным тренировкам.

Технологический стек
Backend: Python 3, FastAPI, Uvicorn
Frontend: HTML, CSS, шаблонизатор Jinja2
База данных: SQL (используется собственная обертка для взаимодействия с БД)
Валидация данных: Pydantic
Безопасность: hashlib, bcrypt
